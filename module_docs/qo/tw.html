qo.tw.goto_definition(obj, command_template='open -na "PyCharm.app" --args --line {lineno} "{filepath}"')[source]

Opens the definition of the object in the file it was defined at the line it was defined.

The default is set to use pycharm to open the file on a mac.

To customize for your system/file_viewer, just use functools.partial to fix the command_template for your case.

For pycharm on other systems, see: https://www.jetbrains.com/help/pycharm/opening-files-from-command-line.html

For vi, see: https://www.cyberciti.biz/faq/linux-unix-command-open-file-linenumber-function/

qo.tw.mk_code_aggregate_md(pkgs, *, copy_to_clipboard=True, save_to_path=None)[source]

Make a markdown file that aggregates all the code in the given packages.

Returns the markdown string.

Optionally (by default) copies the markdown to the clipboard.

If, additionally, you specify a save_to_path (folder or explicit file path), it will save the markdown there too.
